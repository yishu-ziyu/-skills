# 学术论文指引

基于复旦大学熊浩老师在B站讲授的《论文求生指南》课程制作的 Claude Code Skill。

## 安装

```bash
# 克隆仓库
git clone https://github.com/yishu-ziyu/-skills.git

# 复制到 Claude Code skills 目录
cp -r 学术论文指引 ~/.claude/skills/
```

## 核心内容

| 模块 | 内容 |
|------|------|
| 选题三问 | 真问题、好问题、小问题 |
| 论文主脉 | 研究目的 → 研究问题 → 结论 |
| 正文结构 | 分类、逻辑、要素化 |
| 引证规范 | 四大原因、同类一致性 |
| 学术伦理 | 无伤害原则 |

## 一、范文分析：小学语文教材插图的使用研究

### 学生投稿与筛选

我在之前在微博上发了一个通知，就是如果大家有想让我们来看来分析，来帮助大家做修正和提高的论文，可以发给我们。我大概在我的邮箱当中收到了将近30篇的文章。我在其中认真的挑选了一些，我必须在这里向各位同学表达感谢。就是你们的文章成为了所有人批判的素材，这不对，不应该这样讲，所有人可以进一步的阶梯。

我们在这里分析你们的文章，并且发现了其中的问题，目的不是要否定它，目的永远是善良而真诚的。希望大家能够在学习的过程中获得进步。你可以往积极的方向想，对吧？

### 题目分析

**原文题目**：小学语文教材插图的使用研究

**小黄点评**：题目可以再小一点，再具体一点。

**熊浩点评**：

我们在之前给大家谈过选题，一个选题是不是一个好的主题，它应该符合什么条件？

- 它要是一个**真问题**
- 它要是一个**小问题**
- 它要是一个**好问题**（有理论意义）

这个题目：
- ✓ 真问题：插图有没有用？怎么用？这是教学中真实存在的问题
- ✓ 小问题：只聚焦小学语文课本这一类，不是所有插图
- ? 好问题：需要看文章具体怎么论述

### 范文结构分析

文章结构：
- 前言（研究背景 + 研究意义：理论意义+现实意义）
- 研究内容

**熊浩发现硬伤**：论文没有页码，翻起来很麻烦。

### 研究内容的问题

我们看一下他这个文本一开始的前面的部分，前言这个部分，它分成两个部分，研究的背景和研究的意义。

小黄读研究内容部分：
> "它是在基于这个教材的基本研究阐释的基础之上，总结了小学语文教材插图的特点和作用。其次通过跟踪调查对当前小学语文教师教材插图使用现状进行观察。在此期间笔者对教师和学生进行了定时定量的访谈调查的问卷调查。他最后分析了小学在语文教学的过程中如何对插图加以使用，最后提出了一些相关的改良的策略。"

**熊浩评**：你读完这个部分关于内容的阐释，你会发现这篇文章并没有提出研究的问题。

**主题 ≠ 问题**

小学语文教材插图的使用研究，这是一个主题（topic），不是问题（question）。

小学语文教材插图的使用，它具体可以延伸出大量的具体问题：
- 使用什么插图？
- 什么时候会用插图？
- 插图用得多还是少？
- 用插图的效果是好还是不好？
- 学生喜欢用插图还是不喜欢？
- 有插图和没插图有什么差别？

所以你研究什么问题，这篇文章中是需要清晰加以勾勒。但是它只讲了大概要做什么，但 **what is your question？**

正是这篇文章在问题部分的松散，导致这篇文章在结构部分的模糊。

## 二、研究的主脉结构：三角形

我们回到现在我们这个课程当中所呈现的这个三角形，这个三角形就是我所说的我们对整个文章进行分析的这个主体脉络。

他的开始是研究的目标，研究的目的，为什么要做？接下来是研究的问题，而最后是研究的结论。

```
        研究目的
         /    \
        /      \
       /        \
      /   研究   \
     /   问题    \
    /      |     \
   /       |      \
  ←——————— —————————→
```

### 三个核心部分

**1. 研究目的（为什么做？）**
- 现实生活中能不能解决你的爱与痛？
- 理论的意义上能不能帮助你去思考崭新问题？

**2. 研究问题（研究什么？）**
- 基于你感兴趣的领域凝练成主题
- 基于你的主题凝练出具体问题
- 一般一两万字或两三万字的论文，能回答2-3个问题就足够了
- 想回答56789个问题，就是"脑子有问题"

**3. 研究结论（答案是什么？）**
- 直接面对问题，勇敢去回答
- 一定要清晰、有立场

### 三个部分的关系

一旦这三个部分有任何一个部分相对表现比较松散，没有被清晰地厘清，整个文章相对来说就是散的。

这篇文章比较大的问题：
- 选题可以再细（A vs A+的差别）
- 但真正要弥补的是**内在结构**，他没有针对性地去提出问题，以至于他的结论一定模糊

不信？翻到最后一页看他的结论：
> "插图是个重要的资源，它是有意义的，那这个插图可以给学生不一样的体验跟感受，可以让教学的过程丰富多彩。然后我们在过程调查的过程中，也发现了一些问题。然后我们提供一些策略非常空泛。"

所以一旦论证的框架涣散，他最后整理论文的整个结构也就会模糊。

## 三、论文卡住的三个原因

当你在写作论文的过程中感觉到困惑写不下去，很大的概率是这三个维度当中的一个出现了困扰：

| 卡住的地方 | 具体表现 |
|------------|----------|
| **研究目的** | 想不出为什么要研究，这个事儿跟我有半毛钱关系吗？ |
| **研究问题** | 研究动机有了，非常想去研究X，可是研究X什么呢？又卡住了 |
| **研究结论** | 问题能提出来，但回答不了，没有足够的经验和洞察力 |

所以不要抽象地抱怨自己卡在写论文的困境中，而是要清晰地找到是目的问题还是问题问题还是结论问题。

当把自己的困境做了非常清晰地标定以后，我相信你的论文就更容易完成。

## 四、熊浩与小黄的问答：结论怎么写

### 学生常见写法

熊老师，我这堂课给我们讲，你说问题要有结论，我看了非常多的论文，我的结论能不能那么写？

> "我研究了非常多的文献和做了非常多的实验，结果我发现这件事情有好的有不好的。所以我们要科学的、辩证的、合理的来看待这个问题。剩下的有待其他研究者去研究。这个就是我的结论。"

### 熊浩评

对，这种论文就要挂。

你们在写什么学年论文、课程论文的时候，经常写这种东西。对你要知道，你这个论文最多是个**报告**，它其实不算是一个**文章**，文章一定得要有立场。

你研究完了东方，研究完了西方，你最后告诉我说"有精华有糟粕"。那立场很鲜明，不是你可以辨证，用你的话说也可以科学合理。

但如果你说有精华，你必须告诉我什么是精华。有糟粕，你必须告诉我什么是糟粕。我就是要知道哪些是精华，我就是要知道哪些是糟粕，我就是要知道怎么想、怎么改、怎么办。

你说有精华有糟粕，那我看个毛线，我不看你这个文章，我也知道有精华有糟粕。

### "著书立说"的重要性

不要误会我的意思，老师的意思不是说文章中的观点一定要极端，一定全是精华，全是糟粕。

**但你必须给我清晰的结论**，你必须要告诉我：
- 怎么想？
- 怎么办？
- 怎么干？

否则就是对读者的巨大的不负责任。

有学者说中国人写的很多博士论文，叫做**"著书而不立说"**。就是写的跟教科书一样，所有事他都讲了，概念、性质、特征、分类、历史、古今中外他全讲了，他没有**立说**，就是作者在这个地方清晰地表达怎么想、怎么办、怎么干的态度，他没有，这就不能算是一个好的研究。

所以当我们说文章这一讲，我们特别强调文章要有结论的时候，我们是在讲不是一个形式上的结论，而是你得给我们真正的**洞察**。

## 本课小结

| 要素 | 内容 |
|------|------|
| **研究主脉三角形** | 研究目的 → 研究问题 → 研究结论 |
| **常见结构问题** | 只有主题没有具体问题，导致结论模糊 |
| **问题要聚焦** | 一个主题可以延伸出大量问题，选题要具体 |
| **结论要有立场** | 不能是"有精华有糟粕"这种废话，要告诉读者洞察 |
| **论文卡住的原因** | ①研究目的不明 ②研究问题不清 ③没有能力得出结论 |

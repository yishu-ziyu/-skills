## 十一、优秀论文自检流程

```mermaid
flowchart TD
    START[开始写论文] --> A[选题检查]
    A --> B{符合真问题?}
    B -->|否| A1[重新选题]
    B -->|是| C{符合小问题?}
    C -->|否| A2[缩小范围]
    C -->|是| D{有好问题潜力?}
    D -->|否| A3[寻找理论视角]
    D -->|是| E[写导论]
    E --> F[研究目的明确?]
    F -->|否| E1[明确目的]
    F -->|是| G[研究问题清晰?]
    G -->|否| G1[提炼问题]
    G -->|是| H[正文结构]
    H --> I{分类/逻辑/要素?}
    I -->|否| H1[重构结构]
    I -->|是| J[引证规范]
    J --> K{同类格式统一?}
    K -->|否| J1[统一格式]
    K -->|是| L[学术伦理]
    L --> M{有伦理审查?}
    M -->|否| M1[补充审查]
    M -->|是| N[结论]
    N --> O{问题与结论匹配?}
    O -->|否| N1[修正结论]
    O -->|是| END[提交论文]

    style START fill:#9f9,stroke:#333
    style END fill:#9f9,stroke:#333
    style A1 fill:#f99,stroke:#333
    style A2 fill:#f99,stroke:#333
    style A3 fill:#f99,stroke:#333
    style E1 fill:#f99,stroke:#333
    style G1 fill:#f99,stroke:#333
    style H1 fill:#f99,stroke:#333
    style J1 fill:#f99,stroke:#333
    style M1 fill:#f99,stroke:#333
    style N1 fill:#f99,stroke:#333
```

# 学术论文指引 Skill

本skill基于复旦大学熊浩老师在B站讲授的《论文求生指南》课程，共12课内容。

## 课程核心框架

### 选题三问

判断选题质量的三个标准：

| 问题 | 核心 | 自问 |
|------|------|------|
| **真问题** | 有现实痛感 | 这个问题在你生命中留下过痕迹吗？ |
| **好问题** | 有理论意义 | 能否抽象到更高层面？ |
| **小问题** | 可操作 | 能否聚焦到一个具体案例？ |

**选题常见错误**：
- 从众心理——"大家都在研究，我也想研究"
- 缺乏内在热情——没有真实痛感
- 只有现实意义，没有理论野心

### 论文主脉三角

导论和结论都必须锚定这个三角：

```
研究目的 ────→ 研究问题 ────→ 结论
(为什么写?)   (回答什么?)   (答案)
```

**常见问题**：有目的、有结论，但缺乏清晰的研究问题

### 正文结构三种方式

| 方式 | 适用场景 | 示例 |
|------|----------|------|
| **分类** | 按类型分组 | 宫廷画 vs 文人画 |
| **逻辑** | 因果/解决问题 | 分析问题→解决方案 |
| **要素化** | 核心要素构成 | 冲突四要素：利益/选项/标准/替代方案 |

### 引证四大原因

1. **显摆** — 展示专业知识储备，让读者相信你是领域专家
2. **尊重** — 不窃取他人劳动成果
3. **证据** — 支撑自己的论点（数据、引用都是证据）
4. **效率** — 区分新旧知识，提升学术协作效率

**什么时候需要引证？**（When in doubt, cite）
- 不是你第一手做出来的 → 需要引证
- 任何怀疑是否需要引证 → 需要引证
- 不是众所周知的事实 → 需要引证

**同类一致性**：同一类型的引证（期刊/专著/网络资料）格式必须完全一致

### 学术伦理：无伤害原则

**伦理 ≠ 道德**：伦理关乎行业信任机制，有时与道德不同

**伦理委员会审查什么？**
- 不审查研究内容
- 审查研究过程是否对受访者产生伤害
- 保密性、风险最小化

**介入观察的辩证**：
- 短期介入会扭曲观察对象
- 长期田野介入（"演一年"）可消解这种影响

## 使用指南

### 回答用户问题

当用户询问论文写作相关问题时：
1. 先判断属于哪个板块（选题/写作/规范/复习）
2. 调用对应的核心知识点回答
3. 如需详细解释，引用课程具体案例

### 辅助用户写作

当用户需要实际写作帮助时：
1. 帮助梳理选题是否符合三问标准
2. 检查论文主脉是否清晰（目的-问题-结论）
3. 审核引证格式是否规范统一
4. 提示可能的伦理风险

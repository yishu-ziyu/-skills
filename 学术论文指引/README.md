# 学术论文指引

基于复旦大学熊浩老师在B站讲授的《论文求生指南》课程制作的 Claude Code Skill。

## 功能

- 解答选题三问（真问题/好问题/小问题）
- 论文主脉结构指导（目的-问题-结论）
- 引证规范与学术伦理
- 辅助论文写作与自查

## 安装

### 方式一：复制到本地 skills 目录

```bash
cp -r 学术论文指引 ~/.claude/skills/
```

### 方式二：在 Claude Code 中使用

```bash
/plugin marketplace add 你的用户名/学术论文指引
```

## 核心框架

### 选题三问

| 问题 | 核心 |
|------|------|
| 真问题 | 有现实痛感 |
| 好问题 | 有理论意义 |
| 小问题 | 可操作 |

### 论文主脉三角

```
研究目的 → 研究问题 → 结论
```

### 引证四大原因

1. 显摆（展示专业）
2. 尊重（不窃取他人成果）
3. 证据（支撑论点）
4. 效率（区分新旧知识）

### 学术伦理

- **无伤害原则**：研究不能给受访者造成损害
- **同类一致性**：同一类型引证格式必须统一

## 课程内容

共12课，详见 `references/` 目录：

- 选题（真问题、好问题、小问题、方法论切割）
- 写作（导论、文献综述、论证、主脉结构）
- 规范（引证、学术伦理）
- 复习（总结、自查清单）
